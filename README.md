###TODO list web application
Written on Java (Quarkus) and plain HTML and JS
##Running the application
-Make sure you have JDK 21, Quarkus and MongoDB installed
-Make a "tasks" database with "tasks" collection
-Run MongoDB
-Run `quarkus dev` to start the application
-Open `todolist.html` (`src/main/resources/META-INF/resources`) in your browser
